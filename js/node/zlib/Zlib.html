package js.node.zlib

class Zlib

extends Transform<Zlib>

extended by Deflate, DeflateRaw, Gunzip, Gzip, Inflate, InflateRaw, Unzip

Not exported by the zlib module. It is documented here because it is the base class of the compressor/decompressor classes.

Methods

flush (callback:Void ‑> Void):Void

flush (kind:Int, callback:Void ‑> Void):Void

Flush pending data.

kind defaults to Zlib.Z_FULL_FLUSH.

Don't call this frivolously, premature flushes negatively impact the effectiveness of the compression algorithm.

params (level:Int, strategy:Int, callback:Void ‑> Void):Void

Dynamically update the compression level and compression strategy. Only applicable to deflate algorithm.

reset ():Void

Reset the compressor/decompressor to factory defaults. Only applicable to the inflate and deflate algorithms.
