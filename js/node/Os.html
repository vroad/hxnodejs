package js.node

class Os

Provides a few basic operating-system related utility functions.

Static fields

static read only EOL:String

A constant defining the appropriate End-of-line marker for the operating system.

Static methods

staticarch ():String

Returns the operating system CPU architecture.

staticcpus ():Array<CPU>

Returns an array of objects containing information about each CPU/core installed: model, speed (in MHz), and times (an object containing the number of milliseconds the CPU/core spent in: user, nice, sys, idle, and irq).

staticendianness ():Endianness

Returns the endianness of the CPU. Possible values are "BE" or "LE".

staticfreemem ():Float

Returns the amount of free system memory in bytes.

statichostname ():String

Returns the hostname of the operating system.

staticloadavg ():Array<Float>

Returns an array containing the 1, 5, and 15 minute load averages.

The load average is a measure of system activity, calculated by the operating system and expressed as a fractional number. As a rule of thumb, the load average should ideally be less than the number of logical CPUs in the system.

The load average is a very UNIX-y concept; there is no real equivalent on Windows platforms. That is why this function always returns [0, 0, 0] on Windows.
