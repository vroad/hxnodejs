package js.node

class Path

This module contains utilities for handling and transforming file paths. Almost all these methods perform only string transformations. The file system is not consulted to check whether paths are valid.

Static fields

static read only delimiter:String

The platform-specific path delimiter, ; or ':'.

static posix:PathModule

Provide access to aforementioned path methods but always interact in a posix compatible way.

static read only sep:String

The platform-specific file separator. '\' or '/'.

static win32:PathModule

Provide access to aforementioned path methods but always interact in a win32 compatible way.

Static methods

staticbasename (p:String, ?ext:String):String

Return the last portion of a path. Similar to the Unix basename command.

staticdirname (p:String):String

Return the directory name of a path. Similar to the Unix dirname command.

staticextname (p:String):String

Return the extension of the path, from the last '.' to end of string in the last portion of the path. If there is no '.' in the last portion of the path or the first character of it is '.', then it returns an empty string.

staticformat (pathObject:PathObject):String

Returns a path string from an object, the opposite of Path.parse above.

staticisAbsolute (path:String):Bool

Determines whether path is an absolute path.

An absolute path will always resolve to the same location, regardless of the working directory.

staticjoin (paths:Rest<String>):String

Join all arguments together and normalize the resulting path.

staticnormalize (p:String):String

Normalize a string path, taking care of '..' and '.' parts.

When multiple slashes are found, they're replaced by a single one; when the path contains a trailing slash, it is preserved. On Windows backslashes are used.

staticparse (pathString:String):PathObject

Returns an object from a path string.

staticrelative (from:String, to:String):String

Solve the relative path from from to to.

staticresolve (to:String):String

staticresolve (args:Rest<String>):String

staticresolve (from:String, to:String):String

Resolves to to an absolute path.

If to isn't already absolute from arguments are prepended in right to left order, until an absolute path is found. If after using all from paths still no absolute path is found, the current working directory is used as well. The resulting path is normalized, and trailing slashes are removed unless the path gets resolved to the root directory.
